# 📖 Leaderboard<a name="about-project"></a>
>
 Leaderboard is website which displays scores submitted by different players. It also allows you to submit your score. All data is preserved thanks to the external Leaderboard API service. It is built using pure javascript(ES6) , HTML, CSS, and Webpack.

## 💻 Getting Started <a name="getting-started"></a>


### Prerequisites

In order to run this project you need: Configure your code editor with HTML , CSS & JS and some other important extensions



### Setup

git clone git@github.com:zdnahom/leaderboard.git



### Install

Install this project with: 
  
  git clone git@github.com:zdnahom/leaderboard.git


### Usage

To run the project, execute the following command:
  - cd leaderboard
  - npm install
  - npm run build
  - npm start
